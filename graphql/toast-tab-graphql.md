# Toast GraphQL API

Toast is a restaurant technology platform covering POS, ordering, payroll, and delivery. The API covers orders, menus, discounts, labor, payments, and restaurant configuration for enterprise restaurant chains.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Toast REST API
**Documentation:** https://doc.toasttab.com/

- Reference: https://doc.toasttab.com/openapi/
